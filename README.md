# Ejercicios BDD

Dentro de este repositorio están distintos problemas de base de datos antes vistos (o no), su tarea si decide aceptarla es realizar el modelo del problema como ejercicio de normalización y contrastar con el modelo dado, luego plantear su SQL en la forma más contenida posible y nuevamente contrastar con el problema dado.

Finalmente realizar los procedimientos solicitados debajo de los modelos.

Para ambas tablas generar 2 consultas sql que se enviarán como vistas almacenadas, estas deben ser complejas, al menos 1 subconsulta, agrupación y filtro, pueden poblar a gusto para probar.

Para el envío existen 2 opciones, un dump de la base completa como un .sql o un Apellido_Nombre.zip que contega dentro Modelo.sql, Procedimientos.sql, Triggers.sql y Vistas.sql, si se usó exactamente el mismo modelo no es necesario enviar el Modelo.sql, el resto se mantiene.

Se puede enviar un Registros.sql si se gusta para asegurar que funcione con los datos entregados.

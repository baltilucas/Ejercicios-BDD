# Ejercicios BDD

Dentro de este repositorio están distintos problemas de base de datos antes vistos (o no), su tarea si decide aceptarla es realizar el modelo del problema como ejercicio de normalización y contrastar con el modelo dado, luego plantear su SQL en la forma más contenida posible y nuevamente contrastar con el problema dado.

Finalmente realizar los procedimientos solicitados debajo de los modelos.
